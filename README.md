# 飞书 Oncall PE 调优助手 (Feishu Oncall PE Optimizer)

这是一个专为飞书生态打造的 Oncall 自动化处理助手。针对 ToB 客户每季度 200+ 的 Oncall 及售后需求，我们通过扣子（Coze）搭建了高效的自动化处理链路。本助手能够根据用户提交的生图提示词内容和具体的 Oncall 问题，快速给出初步的专业回答和调优方案，极大提升团队响应效率。

## 🌟 核心功能

- **自动化初步响应**：依托扣子（Coze）构建的自动化链路，面对高频的 ToB 售后需求，实现生图 Badcase 和 Oncall 问题的秒级初步诊断与回答。
- **Badcase 智能分析**：深度适配 Seedream 模型的生图逻辑，接收用户提交的「Badcase 内容」，自动提取 Badcase 类型与问题描述。
- **中英双语自适应**：自动识别用户输入的语言（中文/英文），若为英文则全程用英文回复，确保跨国团队及多语言场景下指令精准性。
- **三段式提示词优化**：将提示词严格规范为「前置生图任务提示词 + 主体描述 + 画面风格 & 构图提示词」，表述为正向描述，无否定类表述和括号。
- **极简复用模板**：生成带占位符、字数严格控制在 150 字以内的可复制提示词模板，确保高效复用。
- **Oncall 流程闭环**：支持判定「可直接调优」与「需转接」场景，不可调优时提供明确的对接人指引，支持飞书机器人通知触发，保障问题高效闭环。

## 🚀 业务价值

- **提效降本**：有效应对每季度 200+ 的大客户 Oncall 数量，减少人工初步排查的重复劳动。
- **规范标准**：统一了生图提示词的调优标准，使得跨团队沟通和模型优化有据可依。
- **无缝集成**：完美嵌入飞书工作流，结合扣子（Coze）的强大编排能力，实现“发现问题 -> 提交机器人 -> 获取初步方案 -> (必要时)转人工”的完整闭环链路。
