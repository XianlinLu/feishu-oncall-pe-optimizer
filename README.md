# 飞书 Oncall PE 调优助手 (Feishu Oncall PE Optimizer)

这是一个为 Trae/OpenClaw 工作区打造的专属 Skill（技能），主要用于飞书 Oncall 场景下，深度适配 **Seedream 模型的生图提示词调优**。

## 🌟 核心功能

- **Badcase 智能分析**：接收用户提交的「Badcase 内容」，自动提取 Badcase 类型与问题描述。
- **中英双语自适应**：自动识别用户输入的语言（中文/英文），若为英文则全程用英文回复，确保跨语言场景下指令精准性。
- **三段式提示词优化**：将提示词严格规范为「前置生图任务提示词 + 主体描述 + 画面风格 & 构图提示词」，表述为正向描述，无否定类表述和括号。
- **极简复用模板**：生成带占位符、字数严格控制在 150 字以内的可复制提示词模板，确保高效复用。
- **Oncall 流程闭环**：支持判定「可直接调优」与「需转接」场景，不可调优时提供明确的对接人指引，支持飞书机器人通知触发，保障国内外 Oncall 问题高效闭环。

## 🚀 如何使用

在 Trae/OpenClaw 工作区中加载该技能后，当遇到生图 Badcase 场景时，可以直接在对话框中唤起该助手。

**示例指令**：
```text
/skill feishu-oncall-pe-optimizer 帮我调优一下这个Badcase：主体是一个女孩，但是生成的图片肢体有点崩坏，原来的提示词是：一个女孩在跑步，不要出现多余的手指，高清
```
